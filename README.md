# 亲属关系计算器 (Kinship Calculator)

一个简单、直观的亲属关系计算器，旨在帮助用户轻松理清复杂的中国亲属称谓。

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live%20Demo-brightgreen)](https://hfgong.github.io/qinshu/)

## ✨ 功能

*   **链式计算**：通过依次点击“爸爸”、“的”、“妈妈”等按钮，构建关系链。
*   **实时显示**：每一步操作都会立即显示当前关系链和最终的称谓结果。
*   **丰富的关系库**：内置了从直系亲属到旁系亲属，再到多层姻亲和夫家/妻家亲属的广泛称谓。
*   **简洁的界面**：采用现代、美观的 UI 设计，操作直观，易于上手。
*   **纯静态部署**：无需任何后端服务，可直接托管在 GitHub Pages 或任何静态网站托管平台。

## 🚀 如何使用

1.  打开 [在线演示地址](https://hfgong.github.io/qinshu/)。
2.  界面上会显示 “我的”，这是计算的起点。
3.  通过点击下方的关系按钮（如“父”、“母”、“兄”、“妻”等）来延长关系链。
4.  上方的“关系链”区域会显示您输入的过程，下方的“结果”区域会实时显示最终的称谓。
5.  使用“清除”按钮可以重置关系链，回到“我”。
6.  使用“退格”按钮可以撤销上一步操作。

## 💡 如何扩展

本项目是数据驱动的。所有亲属关系的定义都存储在 `script.js` 文件中的 `relationData` 对象里。

如果您想添加或修改某个称谓，只需编辑这个对象即可。例如，要添加“伯父”的妻子是“伯母”，可以这样做：

```javascript
'伯父': { 'w': '伯母', /* ...其他关系 */ },
```
